============
Installation
============

.. toctree::
   :maxdepth: 1
   :titlesonly:

   prerequisites_fr
   pypi_install_fr
   dev_install_fr


Il y a plusieurs façons d'installer QtPyVCP: soit une installation de développement, soit une installation standard
à partir de PyPi. Lequel vous choisissez dépend de la façon dont vous avez l'intention d'utiliser QtPyVCP.

.. Note::

    Quelle que soit la méthode d'installation que vous utiliserez, vous devrez vous assurer que les
    les prérequis listées sur la page :doc:`Prérequis <prerequisites>` sont satisfaits.


Installation standard
----------------

Si vous n'avez pas l'intention de contribuer à QtPyVCP, alors il est préférable d'installer
à partir de PyPi en suivant le guide :doc:`Installation standard <pypi_install_fr>`.


Installation de développement
-------------------

Si vous souhaitez contribuer à QtPyVCP vous devrez suivre le
le guide :doc:`Installation de développement <dev_install_fr>`.
